# Aurora Cinema Web
This is a frontend web application for a cinema management system
where users can browse movies, view showtimes, and book tickets.

## Features
- Browse movies
- View movie details
- View showtimes
- Responsive design
- Modern UI

## Technologies Used
- JavaScript
- HTML
- Boostrap CSS
  
## Installation Process
- Download the repository (or all the files) from the GitHub link.
- Ensure all files are in one folder on your device.
- Open the folder, and simply open the HTML file in any browser (e.g., Microsoft Edge, Chrome, etc.).

Then you can also add a note that no server or extra dependencies are needed since it’s a static HTML/CSS/JS project. Once you write it, I’d recommend testing the steps yourself to confirm no hidden issues.

## Usage
- Locate the index.html file in the project folder.
- Double-click the file, or right-click and open it with your preferred browser (e.g., Microsoft Edge, Chrome, Firefox).
- Once opened, you will see the homepage displaying a list of movies.
- Click on any movie poster to view detailed information such as showtimes, synopsis, and cast.
- Use the browser’s back button to return to the main listing page.
- The site is fully responsive—resize the window or open it on a mobile device to see the design adapt.
